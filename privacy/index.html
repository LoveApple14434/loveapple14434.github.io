Deadliner 隐私政策

（HarmonyOS 版本）
特别提示：本应用为个人开发者开发的本地工具类应用，不强制要求用户注册账号，无云端账户体系，且不会收集任何可识别您个人身份的信息（包括但不限于姓名、手机号码、身份证号码、电子邮箱等）。
关于生成式人工智能服务的合规声明：
本应用提供的 AI 智能建议功能，本质为调用第三方（DeepSeek）开放平台接口的客户端工具。本应用不运营、不自建生成式 AI 服务。使用该功能前，您必须自行通过合法渠道获取并持有已实名认证的 DeepSeek API Key。本应用将通过技术手段校验 API Key 的有效性，以间接核验用户身份。

本隐私政策旨在向您清晰说明 Deadliner（以下简称“本应用”）在 HarmonyOS 设备运行期间，针对您个人信息的收集、使用、共享、存储及保护的具体规则与措施。请您务必在使用本应用前，仔细阅读并理解本政策全部内容。

一、适用范围

本隐私政策适用于 Deadliner HarmonyOS 客户端当前提供的全部功能模块，具体包括但不限于以下服务：

  • 本地任务与习惯管理功能
  • Deadliner AI 功能（需用户自备 API 的文本生成辅助服务）
  • WebDAV 同步功能（需用户自备同步服务器）

二、个人信息的收集与使用

本应用严格遵循“最小必要”原则，仅在实现对应功能所必需的范围内处理相关信息。

1. 本地任务与习惯管理（核心必要功能）

  • 功能用途：支持您在本地完成任务与习惯的创建、编辑、查询及保存。
  • 处理内容：您手动在本地输入的任务文本、时间节点等任务相关数据。
  • 处理方式：所有数据仅存储于您的 HarmonyOS 设备本地，无任何云端上传行为。

2. Deadliner AI（可选功能，需用户自备第三方服务）

本应用仅作为 API 调用工具，不提供、不运营任何人工智能云端服务。

  • 服务提供方：AI 生成服务由 DeepSeek（杭州深度求索人工智能基础技术研究有限公司） 提供。
  • 实名认证与合规责任：
    • 本应用不建立独立的用户账号体系。
    • 根据《互联网信息服务深度合成管理规定》，使用生成式 AI 服务需进行真实身份信息认证。
    • 您在使用本功能时，需自行提供有效的 DeepSeek API Key。该 Key 的获取需在 DeepSeek 开放平台完成严格的手机号实名认证。
    • 本应用通过校验您提供的 API Key 的有效性，依托 DeepSeek 平台的既有认证体系，完成合规性核验。
  • 数据处理：本应用不存储、不分析任何 AI 功能的请求与返回内容。您提交的文本内容仅在您的设备与 DeepSeek 官方服务器之间进行加密传输。

3. WebDAV 同步（可选功能，需用户自备同步服务器）

本应用仅提供协议适配能力，不提供任何云端同步服务。

  • 账号与凭据：服务器账号、密码等凭据由您自行填写，并通过 HarmonyOS 系统级 SecureStorage 加密后存储于您的设备本地。
  • 责任界定：本应用不运营任何 WebDAV 服务器，也不托管任何同步数据。同步数据的安全性完全由您所选服务器及其运营方负责。

三、个人信息的存储

1. 存储地点

  • 本地任务与设置：仅存储于您的 HarmonyOS 设备本地存储空间。
  • WebDAV 同步数据：存储于您自行配置的第三方服务器。

2. 保留期限

  • 本地数据：保留至您手动删除或卸载本应用为止。
  • 敏感凭据：API Key 及 WebDAV 账号信息保留至您在应用内手动删除。

四、用户画像与营销

本应用不进行任何形式的用户画像构建，不开展用户行为分析，不提供广告推荐服务，也不会将您的相关信息用于任何营销推广用途。

五、Cookie 及同类技术的使用

本应用不使用 Cookie、Web Beacon 等任何自动数据收集技术，仅通过本地数据库及 HarmonyOS 系统安全存储模块，保存应用运行所需的必要配置信息。

六、第三方服务责任声明

本应用包含调用第三方服务（DeepSeek API、WebDAV）的功能。您理解并同意：

  • 服务独立性：第三方服务由其相应的运营主体独立提供，不受本应用控制。
  • 内容合规：您在使用 AI 功能时，应自觉遵守法律法规，不得生成违法违规内容。因您个人使用 DeepSeek API Key 生成的内容所产生的法律后果，由您自行承担。
  • 数据安全：第三方服务对数据的收集和处理遵循其自身的隐私政策。建议您在使用前仔细阅读 DeepSeek 及 WebDAV 服务提供方的隐私协议。

七、信息共享、转让与公开披露

本应用默认状态下不共享任何用户信息。仅在您主动使用 AI 或 WebDAV 功能时，数据会发送至您自行配置的第三方服务。应用开发者不参与任何数据的中转。

在任何情况下，本应用及开发者均不会将用户相关信息进行转让或公开披露（法律法规强制要求除外）。

八、个人信息的保护措施

  • 采用 HarmonyOS 系统级 SecureStorage 对 API Key、WebDAV 凭据等敏感配置进行加密存储。
  • 应用内无任何广告、数据统计、用户行为埋点类 SDK，杜绝第三方工具的信息采集。
  • 安全提示：请您务必妥善保管第三方服务的账号、密码及各类访问凭据。

九、个人信息的自主管理

您可在本应用内自主进行以下操作：

  • 随时修改或删除本地任务数据及习惯配置。
  • 按需删除已配置的 WebDAV 服务器账号及同步凭据。
  • 自主清除已保存的 AI 服务 API Key。
  • 卸载本应用将一次性清除设备本地存储的全部数据。

十、未成年人信息保护

本应用适用用户为年满 3 周岁及以上的自然人。本应用不会主动收集未成年人的个人信息。
